# HABITIX - Management Hunian Modern

HABITIX adalah platform manajemen hunian modern yang dirancang untuk mengelola kompleksitas lingkungan perumahan dengan presisi tinggi. Platform ini menyediakan fitur manajemen iuran, pelacakan aset, forum warga, dan pelaporan keluhan dalam satu sistem yang terintegrasi.

## Fitur Utama
- **Manajemen Iuran**: Pengaturan konfigurasi iuran, penagihan otomatis, dan verifikasi pembayaran.
- **Data Warga**: Manajemen basis data warga yang terstruktur berdasarkan blok dan unit.
- **Pelacakan Aset**: Monitor ketersediaan dan peminjaman aset komplek secara real-time.
- **Komunikasi Warga**: Forum interaktif dan sistem pelaporan keluhan untuk meningkatkan transparansi.
- **Dashboard Analitik**: Visualisasi data keuangan dan operasional untuk pengurus komplek.

## Tech Stack
- **Frontend**: React + Vite
- **Styling**: Tailwind CSS + Chakra UI v3
- **Backend**: Supabase (Auth, Database, Storage)
- **Deployment**: Cloudflare Pages / Wrangler

## Pengembangan
1. Install dependensi: `npm install`
2. Jalankan server lokal: `npm run dev`
3. Build produksi: `npm run build`
